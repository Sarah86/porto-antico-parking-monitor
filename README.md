# Porto Antico Parking (Preact)

A lightweight, real-time dashboard for monitoring parking availability at Porto Antico, Genova.

## Features
- **Real-time Updates**: Automatically refreshes every 30 seconds.
- **Bilingual Support**: Toggle between Italian and English.
- **Direct Navigation**: Integration with Google Maps for each parking location.
- **Mobile Responsive**: Built with a "Genova" maritime theme.

## Tech Stack
- **Framework**: [Preact](https://preactjs.com/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **API**: Porto Antico Public API

## Development
```bash
npm install
npm run dev
```
